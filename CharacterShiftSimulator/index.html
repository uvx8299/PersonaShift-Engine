🧠 模組導覽 / Module Guide

角色情境偏移模擬器 是一個 觀察角色在不同語境中偏移行為的互動模組。

Character Shift Simulator is a interactive module for observing contextual behavior change.

  • 📍 第一步：選擇角色與初始狀態
    Step 1: Choose a character and initial condition
  • 📍 第二步：選擇語境條件進行模擬
    Step 2: Select contextual factors to simulate
  • 📍 第三步：觀察偏移歷程與回饋
    Step 3: Observe shifting trajectory and reactions

📌 範例輸入 / Example Input：
角色：SilentCode
情境：在命名者面前是否保留記憶？
