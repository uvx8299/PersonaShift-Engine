PersonaShift Engine 模組展示

Tone Simulation

記憶模擬器
Memory Simulator

Preference Map

GPT Context Link

結合 GPT 回應模型與模擬器語境。

Multi-Agent Empathic Space

模擬多角色在共感語場中的情緒交互。
