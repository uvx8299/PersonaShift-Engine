
# PersonaShift - GPT Prompt 串接模版

📤 本模組可將角色資料（個性、記憶、語氣偏好）轉換為 GPT 可用的角色扮演 Prompt 格式。

## 🔗 功能

- 自動讀取 `characters.json` 檔案中的角色資訊
- 輸出結構化的 Prompt，可直接用於 ChatGPT、Claude、Gemini 等平台
- 支援語氣模擬與記憶上下文植入

## 📝 Prompt 模版內容包含：

- 人格傾向（traits）
- 過往記憶（memory）
- 語氣偏好（tone）

可用於生成互動型對話、敘事冒險與角色扮演模擬

## ✅ 使用方式

1. 上傳至 GitHub 專案根目錄
2. 訪問頁面後選擇角色並生成 Prompt
3. 將輸出內容複製貼入你使用的 LLM 工具中
