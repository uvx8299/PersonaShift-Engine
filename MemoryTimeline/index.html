🧠 模組導覽 / Module Guide

記憶時間軸導圖 是一個 可追蹤角色記憶發展與語感偏移的模擬工具。

Memory Timeline Mapping is a tool for tracking memory shifts and narrative perception.

  • 📍 第一步：輸入角色名稱與初始記憶內容
    Step 1: Enter a character name and their starting memory
  • 📍 第二步：點選「新增事件」以建立記憶節點
    Step 2: Click 'Add Event' to create memory entries
  • 📍 第三步：可視化角色記憶偏移的時間順序
    Step 3: Visualize the timeline of memory evolution

📌 範例輸入 / Example Input：
角色名：LYF-4322
初始記憶：我曾被語控者強制抹除原始語感。
