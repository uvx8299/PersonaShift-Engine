🧠 模組導覽 / Module Guide

語感認知衝突模擬器 是一個 模擬語義混亂與語感認知張力的衝突場域。

Cognitive Conflict Simulator is a field for simulating semantic confusion and cognitive tension.

  • 📍 第一步：設定兩段語義對立語句
    Step 1: Set two conflicting semantic statements
  • 📍 第二步：選擇角色或主體接受干擾
    Step 2: Select agent receiving conflict
  • 📍 第三步：觀察語感混亂與後續調整
    Step 3: Observe confusion and post-processing behavior

📌 範例輸入 / Example Input：
語句A：「語是秩序之始」；語句B：「語為混沌之器」
